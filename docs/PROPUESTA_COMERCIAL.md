# Propuesta Comercial — Sistema de Gestión para Lavado de Autos

> Versión 1.0 | Abril 2026

---

## ¿Qué es este sistema?

**CarWash ESP** es un sistema ERP especializado para negocios de lavado de autos. Cubre desde el punto de venta hasta la gestión de estacionamiento, fidelización de clientes, inventario y control de trabajadores — todo en una sola plataforma.

Desarrollado en **Laravel 10**, probado con **260+ tests automatizados** y listo para desplegarse en **Laravel Cloud** sin configuraciones complejas.

---

## Costo de Hosting (lo paga el cliente directamente)

El cliente contrata y paga su propia cuenta en **[Laravel Cloud](https://cloud.laravel.com)**. Este costo **no está incluido** en la licencia.

### Estimado mensual según tamaño del negocio

| Perfil del negocio | Plan sugerido | Costo estimado/mes |
|---|---|---|
| Carwash pequeño (1 local, uso diurno) | Starter | $10–20 USD |
| Carwash mediano (1–2 locales) | Starter / Growth | $20–40 USD |
| Carwash grande o cadena | Growth | $40–80 USD |

**¿Por qué Laravel Cloud?**
- Sin necesidad de administrar servidores
- Hibernación automática = 0 costo fuera del horario de operación
- Base de datos MySQL gestionada incluida
- Despliegues automáticos desde Git
- SSL, dominio personalizado y backups incluidos

## Proceso de Entrega

1. **Firma de acuerdo** — Se define el plan contratado y se recibe el 50% del pago.
2. **Configuración** — Se instala el sistema en la cuenta Laravel Cloud del cliente (1–3 días hábiles).
3. **Personalización inicial** — Se cargan productos, tipos de vehículo, tarifas y datos del negocio.
4. **Capacitación** — Sesiones en vivo (Zoom/Meet) con el personal del cliente.
5. **Entrega oficial** — Pago del 50% restante. Inicio del año de mantenimiento.
6. **Soporte continuo** — Atención de dudas y corrección de bugs durante 12 meses.
